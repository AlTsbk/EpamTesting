Аренда автомобилей - https://www.autoeurope.ru

1. Аренда атомобиля несколько часов назад.
    Шаги: Зайти на главную страницу сайта https://www.autoeurope.ru/.
        В поле дата получения выбрать сегодняшнюю дату
        а в поле время получения два часа назад относительно текущего времени.
        Нажать кнопку поиск.
    Ожидаемый результат: появляется всплывающее окно с текстом "Не корректное время аренды".

2. Возврат автомобиля раньше начала аренды.
    Шаги: Зайти на главную страницу сайта https://www.autoeurope.ru/,
        в поле дата получения выбрать завтрашний день,
        а в поле время час вперед относительно текущего времени или 23:59 если текущее время больше 23:00.
        В поле дата возврата выбрать завтрашний день,
        а в поле время возврата час назад относительно текущего времени или 00:00 если текущее время меньше 01:00.
        Нажать кнопку поиск.
    Ожидаемый результат: появляется всплывающее окно с текстом "Не корректное время возврата".

3. Возможность добавить дополнительные принадлежности к авто
    Шаги: Зайти на главную страницу сайта https://www.autoeurope.ru/,
        в поле место получения выбрать "Франция, Paris, Paris Orly Airport",
        в поле дата получения завтрашний день, время 10:00,
        в поле дата возврата дата через две недели относительно даты аренды, время 10:00,
        выберать машину класса "Эконом" в строке самая низкая цена,
        нажать кнопку бронировать "Базовый пакет",
        в блоке дополнительные принадлежности добаить кресло для младенцев 1 шт и навигатор 1 шт,
        затем нажать кнопку далее.
    Ожидаемый результат: в блоке данные бронирования будут отображаться дополнительные принадлежности кресло для младенцев 1 шт и навигатор 1 шт.

4. Возможность арендовать автомобиль.
    Шаги: Зайти на главную страницу сайта https://www.autoeurope.ru/,
        в поле место получения выбрать "Франция, Paris, Paris Orly Airport",
        в поле дата получения завтрашний день, время 10:00,
        в поле дата возврата дата через две недели относительно даты аренды, время 10:00,
        выберать машину класса "Эконом" в строке самая низкая цена,
        нажмать кнопку бронировать "Базовый пакет",
        затем нажать кнопку далее,
        затем нажать кнопку продолжить без апгрейда
    Ожидаемый результат: открывается страница с полями для ввода личной информации и платежных реквизитов, сумма в данных бронирования соответствует ожидаемой.

5. Аренда автомобиля без активного чекбокса(Возраст водителя 25-69).
    Шаги: Зайти на главную страницу сайта https://www.autoeurope.ru/,
        заполнить все поля,
        убрать галочку из чекбокса(Возраст водителя 25-69).
        Нажать кнопку поиск.
    Ожидаемый результат:  появляется всплывающее окно с текстом "Укажите возраст". 

6. Изменение места получения.
    Шаги: Зайти на главную страницу сайта https://www.autoeurope.ru/,
        в поле место получения выбрать "Франция, Figari, Figari Airprt".
        Затем в поле где выбирается страна выбрать "Германию".
        Пропустив выбор города попытаться выбрать место возврата.
    Ожидаемый результат: выпадающий список места возврата окажется пустым.    
    

7. Возврат авто в другом месте.
    Шаги: Зайти на главную страницу сайта https://www.autoeurope.ru/,
        поставить галочку в чекбоксе "Возврат в другом месте".
    Ожидаемый результат: появятся поля "Место возврата".

8. Выставить невозможный возраст.
    Шаги: Зайти на главную страницу сайта https://www.autoeurope.ru/,

        убрать галочку из чекбокса "Возраст водителя 25-69",

        после загрузки страницы нажать кнопку назад в браузере,
        и в поле возраст указать 1500,

    Ожидаемый результат:  появляется всплывающее окно с текстом "Возраст не корректен".
